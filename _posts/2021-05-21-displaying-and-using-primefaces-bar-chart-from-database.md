This blog post is about displaying a bar chart from a database in a Jakarta Faces application using PrimeFaces. In the [PrimeFaces user documentation](https://www.primefaces.org/showcase/ui/chartjs/bar/bar.xhtml?jfwid=929d7), the examples use static data to produce different charts using hard coded values.
